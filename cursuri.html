Cursurile disponibile

Proiectul "Primii Pași în Informatică" vă pune la dispoziție 4 cursuri disponibile pentru orice persoană.
Acestea sunt cursuri de programare în limbajul C++, cursuri de matematică, cursuri de web design și cursuri de Microsoft Ofiice.


X

Cursul C++

Cursul C++ este destinat atât elevilor de gimnaziu, cât și celor de liceu care vor să intre în lumea programrării, începând cu limbajul C++. În cadrul acestui curs se vor pune bazele limbajului C++ și totodată se va dezvolta modul de gândire al problemelor. Limbajul C++ se studiază la liceu și este limbajul din care se susține examenul de bacalaureat.
Ședința este de două ore, iar costul acesteia este de 40 de lei. Orele se vor ține on-line pe platforma Zoom.
La acest curs sunt disponibili doi profesori.

X

Cursul HTML și CSS

Cursul de HTML și CSS este destinat oricărei persoane ce dorește să învețe să creeze un site folosind HTML și CSS. Acest curs este destinat atât celor ce au bazele HTML și CSS, cât și pentru cei începători, cu care se va începe de zero. La încetarea orelelor cursantul va putea crea de unul singur de la zero un site, asemenea site-ului acestui proiect. În plus, pe parcursul acestor ore cursantul va învăța și dezvoltarea unui site pentru telefon.
Ședinta este de două ore, iar costul acesteia este de 50 de lei. Orele se vor ține on-line pe platforma Zoom.
La acest curs este disponibil un profesor.

X

Cursul Microsoft Office

Cursul de Microsoft Office este de asemenea destinat oricărei persoane ce dorește să învețe această suită. De asemenea și la acest curs pot venii atât persoane care nu au mai lucrat cu acest program, cât și cei ce au mai avut tangență cu acesta. Nivelul de predare este cel ECDL, examenul specializat pentru Microsoft Office. La finalul orelor cursantul va putea lua acest examen fară probleme. În cadrul cursului se vor studia Mirosoft Word, Excel, PowerPoint și Acces.
Ședinta este de două ore, iar costul acesteia este de 50 de lei. Orele se vor ține on-line pe platforma Zoom.
La acest curs este disponibil un profesor.

X

Cursul de Matematică

Cursul de Matematică este destinat elevilor de gimnaziu, iar pentru cei de clasa a VIII-a punem la dispoziție pregătire pentru examen. Deși proiectul este unul de informatică, punem la dispoziție și cursuri de matematică deoarece matematica este esențială în acest domeniu. Momentan oferim pregătire doar la nivel de gimnaziu, ulterior ne dorim să ne extindem și la nivel de liceu.
Ședinta este de două ore, iar costul acesteia este de 60 de lei. Orele se vor ține on-line pe platforma Zoom.
La acest curs este disponibil un profesor.
